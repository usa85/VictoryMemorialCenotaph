Soldier's Description

This section includes the profile of a soldier. It includes:

  • First Name, Middle Name, Last Name
  • Soldier matricule
  • Rank (may be several)
  • Military Units (may be several)
  • cenotaph in Minneapolis (GPS GIS coordinate)
  • Cenotaph pictures
  • Actual Grave Location (GPS GIS Coordinate)
  • Actual Grave Pictures
  • Last known address in Minneapolis
  • Closest relatives
  • date death
  • date cause
  • link to the various sites

By clicking on any soldier's name under the List of Soldiers, you should be able to outline the soldier's cenotaph location on the map AND load the above description of the selected soldier on this panel. For example:

  • John Middle Doe, Jr
  • 123456
  • Private First Class
  • 123rd bn, 162 Ghost Regt
  • Cenotaph Location: Latitude, Longitude
  • Burrial Site(s):
    1) Verdun France (Lat, Long),
    2) Some New Resting Place
  • Some Street, Minneapolis, MN
  • Last known relative: Susan Doe, M.
  • Died on February 29th, 19xx
  • KA
