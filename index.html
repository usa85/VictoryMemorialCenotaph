List of Soldiers

This section includes the list of soldiers' name who died during WWI as a result of the war effort and with a related link to the description of the soldiers.

An interactive list of 568 soldier's cenotaphs goes here. Rolling over their name activates the cenotaph's marker, and the name of the soldier. Clicking on either the name or the cenotaph's marker activate the content of the database on the right pannel. It further changes the maps with the existing map diplay into a three map display. The existing one, plus a map of the actual grave, and a map of the battle field the soldier was wounded.

Soldier's Description

This section includes the profile of a soldier. It includes:

  • First Name, Middle Name, Last Name
  • Soldier matricule
  • Rank (may be several)
  • Military Units (may be several)
  • cenotaph in Minneapolis (GPS GIS coordinate)
  • Cenotaph pictures
  • Actual Grave Location (GPS GIS Coordinate)
  • Actual Grave Pictures
  • Last known address in Minneapolis
  • Closest relatives
  • date death
  • date cause
  • link to the various sites

By clicking on any soldier's name under the List of Soldiers, you should be able to outline the soldier's cenotaph location on the map AND load the above description of the selected soldier on this panel. For example:

  • John Middle Doe, Jr
  • 123456
  • Private First Class
  • 123rd bn, 162 Ghost Regt
  • Cenotaph Location: Latitude, Longitude
  • Burrial Site(s):
    1) Verdun France (Lat, Long),
    2) Some New Resting Place
  • Some Street, Minneapolis, MN
  • Last known relative: Susan Doe, M.
  • Died on February 29th, 19xx
  • KA
